# 🌸 Flower Shop — e-commerce project

An application for an online flower shop built with **React**, **Vite**, and **TypeScript**.  
This project is developed as a [Rolling Scopes School](https://rs.school/) final task. 

---

## Tech Stack

- **React** 
- **Vite** 
- **TypeScript** 
- **ESLint** 
- **Husky**  
- **Vitest** 

---

## Getting Started

Follow the steps:

1. Clone the Repository<br>
Clone the repositor using Git and navigate into the project folder:

```bash
git clone https://github.com/tffl/flower-shop
```

2. Install Dependencies<br>
You must have Node.js (v18+) installed. After that, run the following command to install all dependencies:

```bash
npm install
```
